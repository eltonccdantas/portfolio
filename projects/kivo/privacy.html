    KIVO collects no data

    See below all the details about how KIVO handles (or rather, doesn't handle) your information.

    Last updated: March 19, 2026

    🔒 Summary: KIVO does not collect, store, transmit, or share any personal data. Everything happens on your device, with no connection to external servers.

    1. Data Collection

    KIVO collects no personal data whatsoever. There is no usage analytics, telemetry, error reporting, advertising SDKs, or tracking of any kind in the application.

    We have no knowledge of who you are, which files you compress, or how you use the app.

    2. File Processing

    All compression is performed entirely on your device, with no external processing. Your files are never uploaded, transmitted, or shared with any server or third party.

    KIVO does not require an internet connection to function. No network requests are made during normal use of the application.

    The original file is never modified or deleted. KIVO reads the source file, creates a compressed copy in the location you choose, and nothing else.

    3. Permissions

    KIVO requests only the permissions strictly necessary to perform compression:

    Android

    • READ_MEDIA_IMAGES: allows selecting image files.
    • READ_MEDIA_VIDEO: allows selecting video files.
    • READ_EXTERNAL_STORAGE (Android 12 and below): allows file selection on older Android versions.
    • WRITE_EXTERNAL_STORAGE (Android 9 and below): allows saving the compressed file on older Android versions.

    iOS / iPadOS

    File access is provided by the system file picker. No broad access to the photo library or storage is requested.

    macOS / Windows / Linux

    File system access is used only to read the file you select via the system file picker and to write the output file to the location you choose.

    No permission is used for any purpose other than those described above.

    4. Children's Privacy

    KIVO does not collect data from anyone, including children under the age of 13 (or the applicable digital consent age in your jurisdiction). The application does not contain features directed at children that require special data handling under laws such as COPPA, GDPR-K, or similar regulations.

    5. Third-Party Services

    KIVO does not integrate with any third-party services, analytics platforms, advertising networks, social media SDKs, or cloud storage providers.

    The application uses only open-source libraries for compression (FFmpeg, flutter_image_compress, iText/pdf) that operate entirely on the device and make no external connections.

    6. Data Security

    Because KIVO never transmits or stores your data outside your device, the risk of a data breach originating from KIVO is zero. Your files remain under your full control at all times.

    7. Your Rights (GDPR / CCPA)

    Because KIVO collects no personal data, there is no stored data to access, correct, export, or delete. All processed files remain on your device and can be managed using your operating system's standard tools.

    Since no personal data is collected or processed, KIVO has no obligations under GDPR, CCPA, or similar data protection regulations.
