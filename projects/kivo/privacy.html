    KIVO collects no data

    See below all the details about how KIVO handles (or rather, doesn't handle) your information.

    Última atualização: 19 de março de 2026

    🔒 Resumo: O KIVO não coleta, armazena, transmite nem compartilha nenhum dado pessoal. Tudo acontece no seu dispositivo, sem conexão com servidores externos.

    1. Coleta de Dados

    O KIVO não coleta nenhum dado pessoal. Não há análises de uso, telemetria, relatórios de erros, SDKs de publicidade nem rastreamento de qualquer natureza no aplicativo.

    Não sabemos quem você é, quais arquivos você comprime nem como você usa o app.

    2. Processamento de Arquivos

    Toda a compressão é realizada inteiramente no seu dispositivo, sem qualquer processamento externo. Seus arquivos nunca são enviados, transmitidos ou compartilhados com servidores ou terceiros.

    O KIVO não exige conexão com a internet para funcionar. Nenhuma requisição de rede é feita durante o uso normal do aplicativo.

    O arquivo original nunca é modificado ou excluído. O KIVO lê o arquivo de origem, cria uma cópia comprimida no local que você escolher e nada mais.

    3. Permissões

    O KIVO solicita apenas as permissões estritamente necessárias para realizar a compressão:

    Android

    • READ_MEDIA_IMAGES: permite selecionar arquivos de imagem.
    • READ_MEDIA_VIDEO: permite selecionar arquivos de vídeo.
    • READ_EXTERNAL_STORAGE (Android 12 e anteriores): permite a seleção de arquivos em versões mais antigas do Android.
    • WRITE_EXTERNAL_STORAGE (Android 9 e anteriores): permite salvar o arquivo comprimido em versões mais antigas do Android.

    iOS / iPadOS

    O acesso a arquivos é fornecido pelo seletor de arquivos do sistema. Não é solicitado acesso amplo à biblioteca de fotos ou ao armazenamento.

    macOS / Windows / Linux

    O acesso ao sistema de arquivos é utilizado apenas para ler o arquivo que você selecionar via seletor de arquivos do sistema e para gravar o arquivo de saída no local que você escolher.

    Nenhuma permissão é utilizada para finalidade diferente das descritas acima.

    4. Privacidade de Crianças

    O KIVO não coleta dados de ninguém, incluindo crianças menores de 13 anos (ou a idade de consentimento digital aplicável na sua jurisdição). O aplicativo não contém funcionalidades direcionadas a crianças que exijam tratamento especial de dados conforme legislações como o ECA, LGPD, COPPA ou GDPR-K.

    5. Serviços de Terceiros

    O KIVO não se integra a nenhum serviço de terceiros, plataformas de análise, redes de publicidade, SDKs de redes sociais ou provedores de armazenamento em nuvem.

    O aplicativo utiliza apenas bibliotecas de código aberto para compressão (FFmpeg, flutter_image_compress, iText/pdf) que operam inteiramente no dispositivo e não fazem conexões externas.

    6. Segurança dos Dados

    Como o KIVO nunca transmite nem armazena seus dados fora do seu dispositivo, o risco de vazamento de dados originado pelo KIVO é zero. Seus arquivos permanecem sob seu total controle a qualquer momento.

    7. Seus Direitos (LGPD)

    Como o KIVO não coleta nenhum dado pessoal, não há dados armazenados para acessar, corrigir, exportar ou excluir. Todos os arquivos processados permanecem no seu dispositivo e podem ser gerenciados pelas ferramentas padrão do seu sistema operacional.

    Caso você resida no Brasil, seus direitos são garantidos pela Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018). Em razão da ausência total de coleta de dados, nenhuma das obrigações previstas na LGPD se aplica ao KIVO, pois não há operação de tratamento de dados pessoais.

    8. Alterações nesta Política

    Caso esta política de privacidade seja atualizada no futuro, a nova versão será publicada juntamente com o lançamento correspondente do aplicativo. A data de "Última atualização" no topo deste documento refletirá a data da revisão.

    O uso continuado do KIVO após uma atualização da política implica a aceitação dos novos termos.

    🔒 Summary: KIVO does not collect, store, transmit, or share any personal data. Everything happens on your device, with no connection to external servers.

    1. Data Collection

    KIVO collects no personal data whatsoever. There is no usage analytics, telemetry, error reporting, advertising SDKs, or tracking of any kind in the application.

    We have no knowledge of who you are, which files you compress, or how you use the app.

    2. File Processing

    All compression is performed entirely on your device, with no external processing. Your files are never uploaded, transmitted, or shared with any server or third party.

    KIVO does not require an internet connection to function. No network requests are made during normal use of the application.

    The original file is never modified or deleted. KIVO reads the source file, creates a compressed copy in the location you choose, and nothing else.

    3. Permissions

    KIVO requests only the permissions strictly necessary to perform compression:

    Android

    • READ_MEDIA_IMAGES: allows selecting image files.
    • READ_MEDIA_VIDEO: allows selecting video files.
    • READ_EXTERNAL_STORAGE (Android 12 and below): allows file selection on older Android versions.
    • WRITE_EXTERNAL_STORAGE (Android 9 and below): allows saving the compressed file on older Android versions.

    iOS / iPadOS

    File access is provided by the system file picker. No broad access to the photo library or storage is requested.

    macOS / Windows / Linux

    File system access is used only to read the file you select via the system file picker and to write the output file to the location you choose.

    No permission is used for any purpose other than those described above.

    4. Children's Privacy

    KIVO does not collect data from anyone, including children under the age of 13 (or the applicable digital consent age in your jurisdiction). The application does not contain features directed at children that require special data handling under laws such as COPPA, GDPR-K, or similar regulations.

    5. Third-Party Services

    KIVO does not integrate with any third-party services, analytics platforms, advertising networks, social media SDKs, or cloud storage providers.

    The application uses only open-source libraries for compression (FFmpeg, flutter_image_compress, iText/pdf) that operate entirely on the device and make no external connections.

    6. Data Security

    Because KIVO never transmits or stores your data outside your device, the risk of a data breach originating from KIVO is zero. Your files remain under your full control at all times.

    7. Your Rights (GDPR / CCPA)

    Because KIVO collects no personal data, there is no stored data to access, correct, export, or delete. All processed files remain on your device and can be managed using your operating system's standard tools.

    Since no personal data is collected or processed, KIVO has no obligations under GDPR, CCPA, or similar data protection regulations.

    8. Changes to This Policy

    If this privacy policy is updated in the future, the new version will be published alongside the corresponding app release. The "Last updated" date at the top of this document will reflect the revision date.

    Continued use of KIVO after a policy update constitutes acceptance of the new terms.
