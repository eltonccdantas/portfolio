    Dolly Proxy collects no data

    See below all the details about how Dolly Proxy handles (or rather, doesn't handle) your information.

    Last updated: April 6, 2026

    🔒 Summary: Dolly Proxy does not collect, store, transmit, or share any personal data. All traffic interception and processing happens entirely on your machine, with no connection to external servers.

    1. Data Collection

    Dolly Proxy collects no personal data whatsoever. There is no usage analytics, telemetry, error reporting, advertising SDKs, or tracking of any kind in the application.

    We have no knowledge of who you are, which traffic you inspect, or how you use the app.

    2. Traffic Processing

    Dolly Proxy is a local HTTP/HTTPS debugging proxy. All traffic interception, inspection, and editing is performed entirely on your machine. Intercepted requests and responses are never uploaded, transmitted, or shared with any server or third party.

    The application does not require an internet connection to function beyond routing traffic you explicitly direct through it. No network requests are made by the application itself during normal use.

    Any traffic you choose to intercept remains under your full control and is stored only in memory during your session. No session data is persisted to disk by default.

    3. Permissions

    Dolly Proxy requests only the permissions strictly necessary to perform its functions:

    macOS

    • Network access: to listen on the proxy port (default: 8080) and route traffic.
    • File system access: to read and write the mitmproxy CA certificate (~/.mitmproxy/) and application configuration.

    Windows

    • Network access: to listen on the proxy port and route traffic.
    • File system access: to read and write the CA certificate and configuration.

    Linux

    • Network access: to listen on the proxy port and route traffic.
    • File system access: to read and write the CA certificate and configuration.

    No permission is used for any purpose other than those described above.

    4. CA Certificate

    To inspect HTTPS traffic, Dolly Proxy uses mitmproxy's CA certificate for TLS interception. This certificate is generated locally on your machine the first time the proxy runs and stored at ~/.mitmproxy/mitmproxy-ca-cert.pem. It is never transmitted anywhere.

    You must explicitly trust this certificate in your operating system or browser for HTTPS inspection to work. This is a deliberate and informed action on your part.

    5. Android Device Features

    The Android device features (ADB setup, CA install, Frida) operate entirely on your local machine and connected devices. No data from your device or its traffic is transmitted to any server.

    The Frida server binary is downloaded from the official Frida GitHub releases on first use and cached at ~/.dolly/frida/. This is the only outbound network request made by the application, and it only occurs when you explicitly use the Android device feature for the first time.

    6. Children's Privacy

    Dolly Proxy does not collect data from anyone, including children under the age of 13 (or the applicable digital consent age in your jurisdiction). The application does not contain features directed at children that require special data handling under laws such as COPPA, GDPR-K, or similar regulations.

    7. Third-Party Services

    Dolly Proxy does not integrate with any third-party analytics platforms, advertising networks, social media SDKs, or cloud storage providers.

    The application uses open-source libraries — mitmproxy, Flutter, Riverpod, and others — that operate entirely on your device and make no external connections during normal use.

    8. Data Security

    Because Dolly Proxy never transmits or stores your data outside your machine, the risk of a data breach originating from Dolly Proxy is zero. All intercepted traffic and configuration remain under your full control at all times.

    9. Your Rights (GDPR / CCPA)

    Because Dolly Proxy collects no personal data, there is no stored data to access, correct, export, or delete. All data remains on your device and can be managed using your operating system's standard tools.

    Since no personal data is collected or processed, Dolly Proxy has no obligations under GDPR, CCPA, or similar data protection regulations.

    Dolly Proxy is free and open source software. The source code is available on GitHub for anyone to review, audit, or contribute to. This transparency ensures that our privacy claims can be independently verified.
